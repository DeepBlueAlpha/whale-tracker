# 🐳 Deep Blue Alpha — Ethereum Whale Intelligence Platform

**Live platform:** [deepbluealpha.io](https://deepbluealpha.io)

Real-time tracking of 26,000+ Ethereum whale wallets. We surface exchange flow, DEX-level buy/sell activity, and wallet clustering patterns as they happen on-chain.

Deep Blue Alpha monitors a broad set of high-activity Ethereum wallets — the addresses that consistently move markets — and surfaces coordinated on-chain activity across independent wallets in real time.

---

## 📊 Live Scoreboard

Every whale-flagged pick we track is scored publicly, win or loss — nothing here is a snapshot, so we don't publish a static win-rate table that can go stale. See the live, continuously-updated scoreboard at [deepbluealpha.io/picks](https://deepbluealpha.io/picks).

Past on-chain activity is not predictive of future price moves. We do not publish price targets, forward-looking calls, or buy/sell recommendations.

---

## 🔑 What We Track

- **Real-time whale wallet feed** — ETH on-chain activity for 26,000+ wallets, updated continuously
- **Wallet clustering & entity attribution** — surfacing coordinated on-chain activity across independent wallets
- **Token flow tracking** — per-token inflow, outflow, and buy/sell ratios, computed live from tracked wallets
- **Exchange flow tracking** — deposits and withdrawals for tracked wallets, logged in real time
- **Public scoreboard** — every whale-flagged pick tracked, win or loss, publicly auditable

## 🏗️ Technical Foundation

- **Backend:** Python, FastAPI, PostgreSQL
- **Data:** Ethereum nodes via Alchemy
- **Delivery:** WebSocket connections, REST API
- **Coverage:** 26,000+ whale wallets, real-time block monitoring
